# Phase 1 Prerequisites

These are tasks that must be completed by the user (not the agent) before Phase 1 implementation begins. Each file contains a walkthrough for one prerequisite.

## Checklist

- [x] **Go version** -- Verify Go 1.25+ is installed (`prereq-go-version.md`)
- [x] **Testify dependency** -- Add `github.com/stretchr/testify` to go.mod (`prereq-testify.md`)
- [x] **Dev certificates** -- Generate local mTLS certificates for integration tests (`prereq-dev-certs.md`)
- [x] **Branch protection update** -- Add coverage check to branch protection rules (`prereq-branch-protection.md`)
- [x] **Local lint setup** -- Verify golangci-lint v2 is installed and working (`prereq-local-lint.md`)

Complete all items before starting Phase 1 Step 1.
